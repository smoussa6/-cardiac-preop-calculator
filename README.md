# Cardiac PreOp Risk Calculator

A web-based perioperative cardiovascular risk stratification tool for clinicians.

## Risk Scores Included  
- **RCRI** (Revised Cardiac Risk Index)  
- **AUB-HAS2** Cardiovascular Risk Index  
- **NSQIP MICA** (Myocardial Infarction or Cardiac Arrest) Estimate  
- **Functional Capacity** (METs) Assessment

## Features  
- Interactive risk calculators with real-time scoring  
- Auto-generated Assessment & Plan for copy/paste into EMR  
- Mobile-friendly design  
- No data stored — all calculations run locally in your browser

## Disclaimer  
This tool is for educational and clinical decision support purposes only.   
It does not replace clinical judgment. The decision to proceed with surgery   
is made by the performing physician and the patient.
